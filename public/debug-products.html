üîß Common Issues & Solutions

1. "Product not found" Error

  • Cause: Trying to access a product ID that doesn't exist
  • Solution: Check the product ID in the URL or use existing IDs (1, 2)

2. Images Not Loading

  • Cause: CORS restrictions on Google Drive
  • Solution: Use the backend proxy: /api/products/image-proxy?url=YOUR_DRIVE_LINK

3. Backend Not Running

  • Cause: Server not started or crashed
  • Solution: Start backend with cd backend && npm start
